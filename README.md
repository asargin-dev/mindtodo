# MindNode

A mind mapping application built with React, TypeScript, and Vite.

## Tech Stack

- React 19
- TypeScript
- Vite
- TailwindCSS
- Radix UI
- Lucide Icons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Mind mapping interface
- Local storage persistence
- Task management
- Node connections
